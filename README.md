## Cloud Forensic Extractor

A GUI-based tool to extract data from cloud storage (Google Drive / OneDrive) using rclone.

## Features
- GUI interface using Tkinter
- Real-time file transfer progress
- Supports Google Drive / OneDrive
- Threaded execution (non-blocking UI)
- Secure authentication via rclone

## Tech Stack
- Python
- Tkinter
- rclone

## Usage
1. Configure rclone
2. Enter remote name
3. Click "List Files" or "Download Files"

## Requirements

Create a folder named "dump" in the folder where the GUI tool is going to be placed and download rclone in that same folder. When starting new always "configure" before listing or downloading.

Link to download rclone: https://rclone.org/downloads/
